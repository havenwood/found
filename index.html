Found

Find the nearest street address with OS X's CoreLocation! (Found is a bit of glue between evanphx's awesome Lost gem to determine latitude and longitude along with Geokit for the street address.)

Installation

Command line:

$ gem install found

Library:

require 'found

Usage

Command line:

$ found
Rugged Trail, Topanga Canyon, CA 90290, USA

Lost command line:

$ lost
[34.10416324080013, -118.59267817253203]

Library:

Found.address
# => Rugged Trail, Topanga Canyon, CA 90290, USA

Requirements

  • OS X (Lost uses CoreLocation)
  • Command Line Developer Tools (Compile Lost)
  • Wifi (CoreLocation)
